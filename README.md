# 💘 Valentine's Website

A fun and simple Valentine's Day website built to ask one important question:  
**"Will you be my valentine?"**

This project is intentionally lightweight and playful, designed to look like it 
was made in a day using only basic web technologies.                            
---

## 🌐 Live Site
https://miravvaitha.github.io/valentines-website/

---

## ✨ Features

- Minion GIF displayed on page load
- Valentine's question displayed prominently
- Two buttons: Yes and No
- Clicking Yes navigates to a celebration page
- The No button moves to a random position on hover and cannot be clicked       
- Celebration page with a "YAY!! 💖" message
- Falling confetti and balloons using simple animations

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Git & GitHub
- GitHub Pages

No frameworks. No libraries. Just simple web fundamentals.

---

## 🚀 Run Locally

Clone the repository:

    git clone https://github.com/MiravVaitha/valentines-website.git
    cd valentines-website

Open the site locally by opening `index.html` in your browser:

    open index.html

(Alternatively, double-click the file.)

## 🌈 How It Works

- The No button is positioned absolutely and moves to a random location whenever
 the user tries to hover over it.                                               - The Yes button redirects the user to yes.html.
- The celebration page uses basic CSS and JavaScript to create falling confetti 
and balloons.                                                                   
---

## 🚢 Deployment

This website is deployed using GitHub Pages.

Deployment steps:

1. Push all project files to the main branch
2. Go to Settings → Pages
3. Set the source to:
   - Branch: main
   - Folder: / (root)
4. Save and wait for deployment

The site will be available at:

https://miravvaitha.github.io/valentines-website/
